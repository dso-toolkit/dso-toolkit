# Image

Afbeeldingen kunnen responsive worden gemaakt via het toevoegen van de `.img-responsive` class. Deze voegt `max-width 100%; height: auto; display: block` toe aan de afbeelding, zodat hij netjes met zijn parent meeschaalt.

Om afbeeldingen, die gebruik maken van `.img-responsive`, te centreren - gebruik `.center-block` in plaats van `.text-center`
