# Scroll Container

<StorybookComponent implementations={["core", "angular", "react"]} />

Wanneer de inhoud van een blok langer is dan de container zelf toont Scroll Container een schaduw randje om aan te geven dat er meer inhoud aanwezig is.

## Hoe te gebruiken

Plaats de scroll container in een element met een vaste hoogte of die niet langer kan worden vanwege een parent element.
