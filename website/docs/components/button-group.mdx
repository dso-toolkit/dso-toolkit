# Button Group

<StorybookComponent implementations={["core"]} variant="default" />

Button Groups zijn een manier om gerelateerde acties of opties te groeperen in een enkele visuele component. Ze helpen gebruikers om snel keuzes te maken en acties uit te voeren zonder te veel ruimte in te nemen op de pagina.
