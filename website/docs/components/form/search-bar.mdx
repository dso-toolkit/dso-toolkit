## Technische opmerkingen

Pas bij een waarde toont de reset-knop, initieel is deze knop er niet.

## Toegankelijkheid

Een input, en dus ook deze search-bar, dient volgens WCAG succescriterium 1.3.1 te worden voorzien van een label.
Een label wordt aan een specifieke input gekoppeld door het gebruik van het `for`-attribuut.
De waarde van het `for` attribuut moet gelijk zijn aan de waarde van het `id` attribuut van de input.

NB. zie ook "Form > Search Bar" waar het label wordt verstrekt als onderdeel van de `.form-group`:
