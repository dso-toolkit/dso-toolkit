# Angular bindings

Als er gebruik wordt gemaakt van Angular heeft deze methode de voorkeur. De Angular bindings bestaat uit een gegenereerde NgModule met Angular Componenten.

Zie de instructies voor HTML/CSS componenten en Web Components. Daarnaast:

```
npm install @dso-toolkit/angular
```

- https://stenciljs.com/docs/custom-elements
- https://stenciljs.com/docs/angular#consumer-usage
