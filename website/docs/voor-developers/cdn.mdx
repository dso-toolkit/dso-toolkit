# CDN

De DSO Toolkit heeft een eigen CDN op `cdn.dso-toolkit.nl`.

Vanaf versie 50.0.0 bevat de CDN meerdere packages. Vóór versie 50.0.0 werd alleen `dso-toolkit` aangeboden.

In de context van de CDN is een `<versie>`:

- Een semver zonder prefix, bijvoorbeeld: `49.0.2` of `50.0.0`.
- `master`: De laatste build van de `master` branch.

We raden aan om applicaties te fixeren op een versie. `master` is een slepende versie en kan nog wel eens onverwachte resultaten geven.

## >=50.0.0

Vanaf versie 50.0.0 moet het volgende template gebruikt worden: `https://cdn.dso-toolkit.nl/<package>/<versie>`.

- `<package>`: Package name, inclusief scope. Bijvorbeeld `dso-toolkit` of `@dso-toolkit/core`.
- `<versie>`: Een versie vanaf 50.0.0.

## <50.0.0

Toen we begonnen met de toolkit was er maar één package, namelijk `dso-toolkit`. Voor releases tot versie 50.0.0 kan je gebruik maken van het volgende template: `https://cdn.dso-toolkit.nl/<versie>`.
