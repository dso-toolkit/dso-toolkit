## Banner (Web Component)

### De property `noIcon` is hernoemd naar `icon`.

Stencil verbiedt het direct instellen van een prop-defaultwaarde op `true` in de @Prop()-decorator omdat props over het algemeen van buitenaf worden beheerd (bijv. via een parent-component of attribuut in HTML). In deze release is de prop `noIcon`, die default op `true` staat, hernoemd naar `icon` die default op `false`staat.
** Web component **
```html
❌
<dso-banner status="danger">

✅
<dso-banner status="danger" icon></dso-banner>
```

** HTML/CSS component **
```html
❌
<section role="alert" class="dso-banner alert-danger">
  <div class="dso-banner-inner">
    <div class="dso-rich-content">
    <!-- inhoud banner -->
    </div>
  </div>
</dso-banner>

✅
<section role="alert" class="dso-banner alert-danger dso-icon">
  <div class="dso-banner-inner">
    <div class="dso-rich-content">
    <!-- inhoud banner -->
    </div>
  </div>
</dso-banner>
```
