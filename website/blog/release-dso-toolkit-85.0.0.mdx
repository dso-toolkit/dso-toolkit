# DSO Toolkit v85.0.0

Deze release bevat **BREAKING** changes voor het volgende component:

- Autosuggest

<!-- truncate -->

## Autosuggest

De property `suggestOnFocus` is verwijderd. Het gedrag dat de voorgaande suggesties opnieuw getoond worden, wanneer de
invoer weer de focus krijgt, is nu het standaard gedrag bij meer dan 1 suggestie.

### Migratiepad

❌

  ><h2 slot="heading">Omgevingsplan Amsterdam</h2>
  <div slot="meta">

  </div>
