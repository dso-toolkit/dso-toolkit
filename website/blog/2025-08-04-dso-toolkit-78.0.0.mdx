# DSO Toolkit v78.0.0 ⚓️

Deze release bevat **Breaking** changes voor de volgende componenten:

- Ozon Content

<!-- truncate -->

## Ozon Content

De Ozon Content `dsoAnchorClick` event handler is komen te vervallen. Het gedrag van deze event handler is ondergebracht in `dsoClick`. Daarnaast is de event handler `dsoClick` uitgebreid met de nieuwe link types `IntIoRef`, `IntRef` en `ExtRef`.

Er is geen uitgestippeld migratiepad. Hoogstwaarschijnlijk wil je als high-level afnemer gebruik maken van Document Component. Ozon Content kan gezien worden als een "Low Level component".
