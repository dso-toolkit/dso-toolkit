# DSO Toolkit NEXT

In deze release is een kleine aanpassing aan Header gedaan: De property `useDropDownMenu` is hernoemd naar `compact`.

<!-- truncate -->

## Header

Alleen de property `useDropDownMenu` is hernoemd, de mogelijke waarden zijn gelijk gebleven.

### Migratiepad

```html
❌
<dso-header useDropDownMenu="always|auto"></dso-header>

✅
<dso-header compact="always|auto"></dso-header>
```
