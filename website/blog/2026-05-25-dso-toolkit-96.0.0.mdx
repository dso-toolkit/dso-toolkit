# DSO Toolkit v96.0.0 🪁

Deze release bevat **breaking** changes voor de volgende componenten:

- Banner (HTML/CSS implementatie)
- Card Grid (HTML/CSS implementatie)
- Card List (HTML/CSS implementatie)

<!-- truncate -->

## Banner (HTML/CSS implementatie)

Het HTML/CSS component `Banner` is in ⌚ v95.0.0 **deprecated**. In deze release is dit component verwijderd.

### Migratiepad

Gebruik het Core component `Banner`.

```html
❌
<section role="alert" class="dso-banner alert-error dso-icon">
  <div class="dso-banner-inner">
    <div class="dso-rich-content">
      <h2>Storingsmelding:</h2>
      <p>
        Op dit moment ervaren wij een storing in de Vergunningcheck. U kunt wel een
        <a href="#" class="download">aanvraag</a>
        of melding indienen.
      </p>
    </div>
    <dso-icon-button variant="tertiary" icon="cross" label="Sluiten"></dso-icon-button>
  </div>
</section>

✅
<dso-banner status="error" icon="">
  <div class="dso-banner-inner">
    <div class="dso-rich-content">
      <h2>Storingsmelding:</h2>
      <p>
        Op dit moment ervaren wij een storing in de Vergunningcheck. U kunt wel een
        <a href="#" class="download">aanvraag</a>
        of melding indienen.
      </p>
    </div>
    <dso-icon-button variant="tertiary" icon="cross" label="Sluiten"></dso-icon-button>
  </div>
</dso-banner>
```

## Card Grid (HTML/CSS implementatie)

Het HTML/CSS component `Card Grid` is in 🦈 v94.0.0 **deprecated**. In deze release is dit component verwijderd.

### Migratiepad

Gebruik het Core component `Card Container` met mode `grid`.

```html
❌
<div class="dso-card-grid">
  <dso-card href="#"></dso-card>
  [...]
</div>

✅
<dso-card-container mode="grid">
  <dso-card href="#">
  [...]
</dso-card-containe
```

## Card List (HTML/CSS implementatie)

Het HTML/CSS component `Card List` is in 🦈 v94.0.0 **deprecated**. In deze release is dit component verwijderd.

### Migratiepad

Gebruik het Core component `Card Container` met mode `list`.

```html
❌
<ul class="dso-card-list">
  <li>
    <dso-card href="#"></dso-card>
  </li>
  [...]
</ul>

✅
<dso-card-container mode="list">
  <li>
    <dso-card href="#">
  </li>
  [...]
</dso-card-container>
```
