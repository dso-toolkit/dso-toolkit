## Breadcrumbs

Het HTML/CSS component `Breadcrumbs` maakte nog ten onrechte gebruik van de class name in enkelvoud: `breadcrumb`.
Daarnaast ontbrak de prefix `dso-` in de class name. Dat hebben we in deze release gecorrigeerd. De class name is
geworden `dso-breadcrumbs`

### Migratiepad

Vervang de class name `breadcrumb` door `dso-breadcrumbs`.

âŒ

```html
<ol class="breadcrumb">
  [..]
</ol>
```

âœ…

```html
<ol class="dso-breadcrumbs">
  [..]
</ol>
```

## Label

Het HTML/CSS component `Label` is deprecated. In een toekomstige **BREAKING** release zal dit component verwijderd
worden.

### Migratiepad

Gebruik de Core implementatie van Label.

âŒ

```html

  <span class="dso-label-symbol">[...]</span>
  LABEL

    <span class="sr-only">Verwijder</span>
    <dso-icon icon="times"></dso-icon>
  </button>
```
