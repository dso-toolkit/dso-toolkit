Zoekbalk gesloten

De zoekbalk dient vanuit zichzelf de class .dso-searchbar te hebben en kan worden geopend met de modifier classes .dso-open. Het toevoegen/verwijderen van deze modifier classes start een animatie die de zoekbalk opent of sluit.

Zoekbalk open

Om een zoekbalk te openen dient de class .dso-open toegevoegd te worden. Sluiten gaat door de class weer te verwijderen
