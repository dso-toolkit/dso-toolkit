Ongeldig

Indien een input ongeldig is, dient er een .dso-invalid class gezet te worden op de div met de class .dso-input.

Ongeldig e-mailadres ingevuld

Bijvoorbeeld: email@email.nl

Geldig

Indien een input ongeldig was en valide wordt gemaakt, dient de .dso-invalid class vervangen te worden door de class .dso-valid.

Bijvoorbeeld: email@email.nl
